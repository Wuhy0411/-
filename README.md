## 📌 项目简介

本项目旨在通过整合底层硬件外设与上位机软件，搭建一套完整、稳定的小鼠滚轮行为学实验平台。该平台能够实时监测、记录并可视化小鼠在滚轮上的运动节律与活动轨迹。

采集到的高精度时间序列数据（如速度、运动距离、启停时间戳等），格式规整，可直接导出参与后续的**动态决策行为分析**或**潜在认知状态建模**，为行为学神经机制研究提供可靠的数据支撑。

---

## 🛠️ 系统架构与环境依赖

本项目由**硬件数据采集端（下位机）**与**软件数据处理端（上位机）**两部分组成。

### 1. 硬件环境 (Hardware)
* **核心控制板：** Arduino 开发板（如 Arduino Uno / Nano）
* **传感器：** 光电编码器 或 霍尔传感器（用于精准记录滚轮转动圈数与方向）
* **机械组件：** 标准小鼠测试滚轮、3D打印固定支架
* **通信线路：** USB 数据传输线

### 2. 软件环境 (Software)
详细的环境安装与配置步骤，请参阅 [《软件开发环境安装指南》](软件安装流程.md)。
* **下位机开发环境：** Arduino IDE
* **上位机开发环境：** VS Code
* **上位机语言与依赖：**
  * Python 3.14 及以上版本
  * `pyserial`：用于与 Arduino 进行串口数据通信
  * `matplotlib`：用于运动数据的实时动态可视化

## 🚀 快速开始 (操作说明)

### 第一步：硬件连接
1. 将光电编码器/传感器正确安装至滚轮转轴处。
2. 将传感器的信号引脚（VCC, GND, OUT）按照针脚定义连接至 Arduino 控制板。
3. 使用 USB 线将 Arduino 连接至电脑。

### 第二步：下位机程序烧录 (Arduino端)
1. 打开 `Arduino IDE`。
2. 导入本项目 `arduino_code/` 目录下的 `.ino` 源码文件。
3. 在菜单栏选择对应的开发板型号（如 Arduino Uno）及正确识别的 COM 端口。
4. 点击 **上传 (Upload)** 按钮，将代码烧录至控制板。
   > *提示：上传成功后，可打开右上角的“串口监视器”转动滚轮，测试是否有原始数据输出。*

### 第三步：上位机数据采集与可视化 (Python端)
1. 打开 `VS Code`，并在终端中激活配置好的 Python 环境。
2. 检查串口号配置：打开 `python_script/main.py`，确认代码中的 `COM_PORT` 变量化为您电脑实际分配给 Arduino 的端口号（如 `COM3`）。
3. 运行主程序开始实验：
   ```bash
   python main.py
   ```
4. **运行现象：**
   * 终端将实时打印小鼠的运动状态数据。
   * `matplotlib` 会弹出一个动态图表窗口，实时绘制小鼠的运动速度与时间关系曲线。

## 📁 数据输出说明

实验结束后，系统会自动在 `data/` 目录下生成一个以当前时间戳命名的 CSV 文件（例如 `wheel_data_20260415_1530.csv`）。

**数据包含以下核心字段：**
* `Timestamp (ms)`: 相对时间戳
* `Revolutions`: 累计圈数
* `Speed (cm/s)`: 瞬时速度
* `Direction`: 运动方向（正向/反向）

> **💡 数据应用建议：** 输出的 CSV 数据结构清晰，非常适合导入至 Python 进行二次清洗，或直接用于构建状态空间模型（如 HMM 系列模型）以分析小鼠的行为切换策略。
