# 硬件安装步骤说明

本文档用于说明硬件组件的准备、接线、滚轮结构安装以及旋转编码器安装流程。请按照步骤依次操作，避免漏装或提前固定导致后续调整困难。

---

## 1. 硬件组成清单

完整硬件应包含以下部件：

| 序号 |             部件名称 | 数量 | 说明                       |
| ---- | -------------------: | ---: | -------------------------- |
| 1    |                 辐条 |    2 | 亚克力透明辐条             |
| 2    |    RS485 转 USB 模块 |    1 | 用于连接旋转编码器与电脑   |
| 3    |               支撑杆 |    2 | 用于支撑滚轮轴             |
| 4    |           旋转编码器 |    1 | 用于采集旋转角度或位置信号 |
| 5    |         编码器固定器 |    1 | 用于固定旋转编码器         |
| 6    |                 轴承 |    2 | 用于支撑转轴旋转           |
| 7    |                 法兰 |    2 | 用于固定辐条与转轴         |
| 8    |                 底座 |    1 | 用于安装整体结构           |
| 9    | 配套螺钉、螺柱、螺母 | 若干 | 包括 M2、M5 等规格         |

## 2. 安装前准备

### 2.1 去除辐条保护贴纸

拆掉辐条表面的保护贴纸。处理完成后，亚克力辐条应呈透明状态。

<img src="images/2.jpg" width="60%" />

---

## 3. 旋转编码器接线

本步骤用于将旋转编码器连接至 RS485 转 USB 模块。

### 3.1 拆除编码器端子线

使用镊子按下接头中间的卡扣，然后轻轻拔出线缆。

<img src="images/3.jpg" width="60%" />

### 3.2 松开 RS485 转 USB 模块端口螺丝

使用平口螺丝刀逆时针拧松端口螺丝，直到能够观察到完整卡扣露出。

<img src="images/4.png" width="60%" />

### 3.3 插入编码器线缆并固定

将编码器线缆插入对应端口，然后顺时针拧紧端口螺丝。

接线对应关系如下：

| 编码器线缆颜色 | RS485 转 USB 模块端口 |
| -------------- | --------------------- |
| 白色线         | A                     |
| 绿色线         | B                     |
| 黑色线         | 0                     |
| 红色线         | 5V                    |
| 黄色线         | 置空，不接            |

<img src="images/5.jpg" width="60%" />

> 注意：拧紧端口螺丝后，可轻轻拉动线缆确认是否固定牢靠。

---

## 4. 滚轮结构安装

### 4.1 固定法兰与辐条

使用 M2 螺柱和 M2 六角螺母，将法兰固定在辐条中心位置。

<img src="images/6.jpg" width="60%" />

---

### 4.2 安装第一侧轴承与辐条

在转轴的合适位置贴一圈双面胶，然后将轴承套在该位置。

由于轴承与转轴之间存在装配公差，双面胶可用于增强轴承与转轴之间的固定效果。

操作顺序如下：

1. 在转轴适当位置贴一圈双面胶。
2. 将轴承套入贴有双面胶的位置。
3. 将轴承与转轴一起放入支撑件孔位中。
4. 使用 M2 螺柱压紧轴承。
5. 将已安装法兰的辐条套入转轴。

<img src="images/7.jpg" width="40%" />

<img src="images/8.jpg" width="60%" />

<img src="images/9.jpg" width="60%" />

注：轴前端到固定器平面的距离最好设置为4.5cm。

<img src="images/19.png" width="60%" />

---

### 4.3 安装第二侧辐条与轴承

将另一个辐条套入转轴，然后将另一个支撑件放到合适位置，此时支撑件暂不固定。

根据轴承的大致安装位置，在转轴对应位置贴一圈双面胶，再将第二个轴承套入转轴并固定。固定方式与步骤 4.2 相同。

操作顺序如下：

1. 将第二个辐条套入转轴。
2. 放置第二个支撑件，暂不固定。
3. 确认轴承的大致位置。
4. 在对应位置贴一圈双面胶。
5. 将轴承套入转轴。
6. 使用 M2 螺柱固定轴承。

<img src="images/11.jpg" width="60%" />

<img src="images/12.jpg" width="60%" />

### 4.4 调整滚轮间距并固定法兰

调整两个滚轮至合适间距后，使用法兰自带的紧定螺丝固定法兰在转轴上的位置。

完成本步骤后，滚轮部分安装结束。

<img src="images/13.jpg" width="60%" />

<img src="images/14.jpg" width="60%" />

> 注意：固定前应确认两个辐条间距合适，且转轴能够顺畅转动。

## 5. 底座与编码器安装

### 5.1 安装底座螺柱、联轴器与套筒

在底板上安装 M5 螺柱。

随后在转轴末端装入联轴器。此时联轴器无需固定，只需先套入转轴即可。

最后将套筒固定在底座上。

<img src="images/15.jpg" width="60%" />

<img src="images/16.jpg" width="60%" />

---

### 5.2 安装旋转编码器与编码器固定器

将旋转编码器、编码器固定器与光杆固定好，然后将其放入套筒中。此时不需要立即固定。

调整编码器高度，使编码器轴能够顺利插入联轴器内部。确认位置合适后，拧紧套筒旋钮。

<img src="images/17.jpg" width="60%" />

---

### 5.3 固定联轴器与编码器

拧紧联轴器，使转轴与编码器轴可靠连接。

最后使用 M2 螺柱固定编码器，完成整体安装。

<img src="images/18.jpg" width="60%" />

---

## 6. 安装完成检查

安装完成后，建议按照以下项目进行检查：

| 检查项目   | 检查要求                         |
| ---------- | -------------------------------- |
| 辐条状态   | 两个辐条安装牢固，无明显晃动     |
| 滚轮间距   | 间距合适，未发生干涉             |
| 转轴转动   | 转动顺畅，无明显卡滞             |
| 轴承固定   | 轴承位置稳定，无明显松动         |
| 法兰固定   | 紧定螺丝已拧紧                   |
| 编码器连接 | 编码器轴已插入联轴器并固定       |
| 接线状态   | A、B、0、5V 接线正确，黄色线置空 |
| 底座结构   | 支撑件、套筒、底座连接稳定       |

---

## 7. 注意事项

1. 编码器接线前，请确认 RS485 转 USB 模块端口螺丝已充分拧松。
2. 接线完成后，应检查线缆是否牢固，避免接触不良。
3. 轴承安装时，双面胶应贴在轴承实际固定位置，避免后续重复拆装。
4. 第二个支撑件在调整阶段不要提前固定，便于后续调整滚轮间距。
5. 联轴器在编码器高度确认前不要提前拧紧。
6. 所有螺丝固定完成后，应手动转动滚轮，确认结构转动顺畅。
