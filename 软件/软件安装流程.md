# 🐍 一、Python 安装与环境配置

Python 用于运行上位机程序、串口通信程序以及后续的数据处理和绘图程序。

---

## 1. 下载 Python 解释器

### 1.1 🌐 进入 Python 官网

打开 Python 官方网站：

<https://www.python.org/>

将鼠标悬停或点击导航栏中的 **Download**。

![](images/图片1.png)

---

### 1.2 📥 选择 Python 版本

在 Download 页面向下滚动，找到所需版本，例如 Python 3.14，点击对应的 **Download** 按钮。

![](images/图片2.png)

---

### 1.3 🪟 下载 Windows 安装包

进入版本详情页后，继续向下滚动，找到 **Files** 列表。

选择 Windows 版本安装包：`Windows installer (64-bit)`。

点击后等待安装包下载完成。

![](images/图片3.png)

---

## 2. 安装 Python

### 2.1 ⚙️ 运行安装包

双击下载好的 Python 安装包，进入安装向导。

请务必勾选底部的两个选项，尤其是 `Add Python to PATH`。

随后点击 `Customize installation`。

![](images/图片4.png)

> ⚠️ 注意：如果没有勾选 `Add Python to PATH`，后续可能会出现命令行无法识别 `python` 或 `pip` 的问题。

---

### 2.2 ✅ 选择可选功能

在 **Optional Features** 页面，确保所有可选功能均已勾选，然后点击 **Next**。

![](images/图片5.png)

---

### 2.3 🔧 设置高级安装选项

在 **Advanced Options** 页面，建议勾选 `Install Python for all users`。

随后点击 **Install** 开始安装，并等待安装完成。

![](images/图片6.png)

---

## 3. 验证 Python 安装

### 3.1 🖥️ 打开命令提示符

按下快捷键 `Win + R`。

在运行窗口中输入 `cmd`，然后按回车键打开命令提示符。

![](images/图片7.png)

---

### 3.2 🔍 检查 Python 是否可用

在终端中输入：

    python

如果能够看到 Python 版本号，并进入 `>>>` 交互界面，则说明 Python 安装成功。

![](images/图片8.png)

---

### 3.3 🚪 退出 Python 交互界面

如果已经进入 `>>>` 交互界面，可输入以下命令退出：

    exit()

---

## 4. 安装 Python 依赖库

后续程序运行需要安装部分第三方库，主要包括：

| 库名称 | 作用 |
| :--- | :--- |
| `pyserial` | 串口通信 |
| `wheel` | 辅助 Python 库安装 |
| `matplotlib` | 数据绘图与可视化 |

---

### 4.1 📋 查看当前已安装库

重新打开一个 cmd 终端，或在退出 Python 交互界面后输入：

    pip list

刚安装完成时，第三方库数量通常较少。

![](images/图片9.png)

---

### 4.2 🔗 安装 pyserial 库

输入以下命令安装 `pyserial`：

    pip install pyserial -i https://pypi.tuna.tsinghua.edu.cn/simple

![](images/图片10.png)

如果终端显示类似 `Successfully installed pyserial`，说明安装成功。

---

### 4.3 🧱 安装 wheel 库

为了保证后续库能够顺利安装，建议先安装 `wheel`：

    pip install wheel

![](images/图片11.png)

如果终端显示类似 `Successfully installed wheel`，说明安装成功。

---

### 4.4 📊 安装 matplotlib 库

输入以下命令安装 `matplotlib`：

    pip install matplotlib

![](images/图片12.png)

如果终端显示类似 `Successfully installed matplotlib`，说明安装成功。

---

### 4.5 ✅ 确认依赖库安装结果

再次输入：

    pip list

检查列表中是否包含以下库：

- `pyserial`
- `wheel`
- `matplotlib`

![](images/图片13.png)

---

# 💻 二、VS Code 安装与配置

VS Code 用于编写、运行和调试 Python 程序。

---

## 1. 下载 VS Code

### 1.1 🌐 进入 VS Code 官网

打开 VS Code 官方网站：

<https://code.visualstudio.com/>

点击首页的 **Download for Windows** 按钮，等待安装包下载完成。

![](images/图片14.png)

---

## 2. 安装 VS Code

### 2.1 ⚙️ 运行安装程序

双击下载好的 VS Code 安装包。

![](images/图片15.png)

---

### 2.2 📄 接受许可协议

勾选接受许可协议，然后点击 **下一步**。

![](images/图片16.png)

---

### 2.3 📁 选择安装路径

选择合适的安装路径，然后点击 **下一步**。

---

### 2.4 📂 确认开始菜单文件夹

保持默认设置即可，点击 **下一步**。

![](images/图片17.png)

---

### 2.5 ✅ 勾选附加任务

建议将“其他”中的选项全部勾选，例如：

- 添加到右键菜单
- 将 Code 注册为受支持文件类型的编辑器
- 添加到 PATH

勾选完成后点击 **下一步**，随后点击 **安装**。

![](images/图片18.png)

---

## 3. 安装 VS Code 插件

### 3.1 🌏 安装中文语言包

打开 VS Code，点击左侧活动栏中的扩展图标。

在搜索框中输入 `Chinese`。

找到中文简体语言包，点击 **Install**。

![](images/图片19.png)

安装完成后，右下角会弹出提示，点击 `Change Language and Restart`。

VS Code 会自动重启并切换为中文界面。

![](images/图片20.png)

---

### 3.2 🐍 安装 Python 插件

再次进入扩展商店，在搜索框中输入 `Python`。

找到 Microsoft 官方提供的 Python 插件，点击 **安装**。

![](images/图片21.png)

安装完成后，VS Code 即可用于 Python 程序编写、运行和调试。

# 🧩 四、CP2102 USB 串口驱动安装

CP2102 驱动用于识别 USB 转串口模块。若设备管理器中已经可以正常看到 CP2102 对应端口，则无需重复安装驱动。

---

## 1. 检查设备状态

### 1.1 🔍 打开设备管理器

连接编码器或 USB 转串口模块后，打开 Windows 设备管理器。

如果看到 `CP2102 USB` 设备旁边存在黄色感叹号，说明驱动尚未正确安装。

<img src="images/图片30.png" width="80%" />

> ✅ 如果已经能在“端口（COM 和 LPT）”中看到 `CP2102 USB to UART Bridge Controller`，则说明驱动已正常安装，可以跳过本章节后续步骤。

---

## 2. 更新驱动程序

### 2.1 🔧 进入驱动更新界面

双击带有黄色感叹号的 `CP2102 USB` 设备。

在弹出的设备属性窗口中，点击 `更新驱动程序`。

<img src="images/图片31.png" width="50%" />

---

## 3. 手动选择驱动文件夹

### 3.1 📁 选择本地驱动目录

在驱动更新界面中，选择 `浏览我的电脑以查找驱动程序`。

然后在目录中选择驱动文件夹：`CP210x_Universal_Windows_Driver`。

选择完成后，点击更新或下一步，等待系统完成驱动安装。

<img src="images/图片32.png" width="50%" />

---

## 4. 确认驱动安装成功

### 4.1 ✅ 检查端口显示

驱动安装成功后，重新打开设备管理器。

在 `端口（COM 和 LPT）` 中查看设备。

如果能够看到类似 `CP2102 USB to UART Bridge Controller` 的端口信息，则说明 CP2102 驱动安装成功。

---

# ✅ 五、安装完成检查

完成所有安装后，建议按照下表进行检查。

| 检查项目 | 检查方法 | 正常结果 |
| :--- | :--- | :--- |
| Python 是否安装成功 | 在 cmd 中输入 `python` | 显示 Python 版本号并进入 `>>>` |
| pip 是否可用 | 在 cmd 中输入 `pip list` | 能正常显示库列表 |
| pyserial 是否安装成功 | 在 cmd 中输入 `pip list` | 列表中包含 `pyserial` |
| matplotlib 是否安装成功 | 在 cmd 中输入 `pip list` | 列表中包含 `matplotlib` |
| VS Code 是否安装成功 | 打开 VS Code | 能正常进入编辑器界面 |
| 中文语言包是否生效 | 查看 VS Code 菜单栏 | 界面显示为中文 |
| Python 插件是否安装 | 查看 VS Code 扩展 | Python 插件显示已安装 |
| Arduino IDE 是否安装成功 | 打开 Arduino IDE | 软件能正常启动 |
| CP2102 驱动是否正常 | 打开设备管理器 | 端口中显示 CP2102 设备 |

# ❓ 六、常见问题处理

## 1. 输入 `python` 后提示不是内部或外部命令

### 可能原因

安装 Python 时没有勾选 `Add Python to PATH`。

### 处理方法

1. 重新运行 Python 安装程序。
2. 选择 **Modify**。
3. 检查并补充 PATH 相关选项。
4. 安装完成后重新打开 cmd。

---

## 2. 输入 `pip` 后无法识别

可尝试使用以下命令代替：

    python -m pip list

安装库时也可以使用：

    python -m pip install pyserial
    
    python -m pip install matplotlib

---

## 3. CP2102 设备仍然显示黄色感叹号

请检查以下项目：

1. USB 线是否支持数据传输。
2. 驱动文件夹是否选择正确。
3. 是否选择了 `CP210x_Universal_Windows_Driver` 文件夹。
4. 是否重新插拔过设备。
5. 是否重新打开设备管理器刷新状态。

---

## 4. Arduino IDE 初次打开时弹出多个驱动安装窗口

这是正常情况。

请对每个弹窗都点击 `安装`，否则可能导致部分开发板或串口设备无法被识别。

---

# 📦 七、推荐安装顺序

建议按照以下顺序进行安装：

1. 🐍 安装 Python。
2. 📦 安装 Python 依赖库。
3. 💻 安装 VS Code。
4. 🌏 安装 VS Code 中文语言包。
5. 🐍 安装 VS Code Python 插件。
6. 🔌 安装 Arduino IDE。
7. 🧩 连接设备并检查 CP2102 驱动。
8. ✅ 必要时手动安装 CP2102 驱动。

完成以上步骤后，即可开始进行软件开发与设备调试。
